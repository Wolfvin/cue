Quick Start

From zero to a running demo in three steps.

1
Install the core package
Add the cue SDK to your project. Zero external dependencies.
# npm
npm install @cue-vin/core

# or pnpm
pnpm add @cue-vin/core
2
Generate a DemoScript
Describe your features in structured JSON and call generate(). No LLM call required — the heuristic engine converts features into a playable demo script automatically.
import { generate } from "@cue-vin/core";

const script = generate({
  id: "my-demo",
  title: "My Product Demo",
  features: [
    {
      name: "Dashboard",
      description: "See key metrics at a glance",
      hotspots: [{ label: "Revenue", x: 0.25, y: 0.3 }]
    },
    {
      name: "Get Started",
      description: "Join 10k+ users",
      cta: { type: "email_capture", label: "Start trial" }
    }
  ]
});
3
Connect to Claude via MCP
Let AI agents generate demos on the fly. Add the MCP server to your Claude Desktop config — your agent can then call generate, validate, and export as tools.
// claude_desktop_config.json
{
  "mcpServers": {
    "cue": {
      "command": "npx",
      "args": [ "@cue-vin/mcp" ]
    }
  }
}
