cue-utils.iife.js
A standalone IIFE bundle exposing Timeline and Pointer from @cue-vin/core — no React, no npm, no module bundler. Just load via <script> tag and use window.CueUtils.
What's Inside
⏱️
CueUtils.Timeline
Schedule callbacks with setTimeout, support looping, auto-advance, and clean disposal.
👆
CueUtils.Pointer
Animate a cursor with moveTo(), click(), and keyframe-based play(). Drives DOM via onChange callback.
📦
2.6 kB Minified
Timeline + Pointer with zero dependencies. Compare: cue-player.iife.js is 180 kB with React.
🌐
Vanilla HTML Ready
Load via <script src="...iife.js"> and access window.CueUtils. No bundler required.
